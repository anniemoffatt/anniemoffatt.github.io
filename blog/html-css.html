What does it mean to Display Inline vs Inline Blocks?

Both are CSS terms that determine how an element behaves (layout wise). Originally I thought they only applied to images, but they can be used on any HTML element.

Display Inline

This means the element will sit in line with other elements. It doesn't start on a new line and you can't set the width or height.

Often used for:

  • Text level elements
  • Keeping things in line with other content
  • Styling small pieces of text without affecting layout (like changing the colour of just one word in a sentence)

Inline Blocks

Similar to Display Inline in the sense it still sits in line with other other elements, but with more flexibility. You can set the width, height, padding and margins.

Often used for:

  • Buttons you might want to size or space out
  • Navigation links that need padding or custom styling
  • Images you want to control the size of
  • Form inputs like text fields or submit buttons

In short, both are ways to control how elements behave on a webpage. Inline is great for small elements that flow naturally with text, but you won't have much control over size. Block provides the side by side layout as well as the ability to to set width, height and spacing.
